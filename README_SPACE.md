# Sehen Lernen - Learning to See

A comprehensive image analysis and computer vision platform for feature extraction, clustering, and similarity search.

## Features

- **Feature Extraction**: 100+ image features (color, texture, shape, edges, deep learning)
- **K-means Clustering**: Visualize and analyze image clusters
- **Similarity Search**: Find similar images in your dataset
- **Statistical Analysis**: Comprehensive image statistics
- **Data Management**: Upload, process, and download results

## Usage

1. Upload images to analyze
2. Select features to extract
3. View results and visualizations
4. Download analysis results

## Technologies

- Streamlit (Frontend)
- FastAPI (Backend)
- OpenCV, scikit-learn, PyTorch
- And many more ML/CV libraries

## About

Sehen Lernen makes computer vision accessible for education and research.
