#### Windows:

#### Windows:
```

```

#### Mac/Linux:
```
1. Open Terminal
2. bash start_app.sh
3. Wait 10 seconds
4. Browser opens to app
```

---

### üìç WHAT TO EXPECT

**Backend Window:**
- Shows lots of logs
- You should see: "Application startup complete"
- Don't close this!

**Frontend Window:**
- Shows fewer messages
- You should see: "Local URL: http://localhost:8501"
- Don't close this!

**Browser:**
- Should open automatically
- If not: Go to http://localhost:8501
- You're ready when you see the interface!

---

### üõë STOPPING

Close both:
1. Backend Command Prompt/Terminal
2. Frontend Command Prompt/Terminal

Or press: Ctrl+C in each window

### ‚ö†Ô∏è QUICK FIXES

| Problem | Solution |
|---------|----------|
| "Python not found" | Install Python from python.org |
| "Port already in use" | Close other applications using ports |
| Frontend keeps reloading | Wait 20 seconds, close browser, reopen |
| Backend not showing | Wait 5 seconds, both windows should open |
| Nothing happens | Try running again, wait longer |

### üí° TIPS

- First run takes longer (loads ML models)
- Smaller images process faster
- Supported formats: JPEG, PNG, BMP, TIFF
- Max file size: 50 MB (1-20 MB recommended)
- All processing happens on your computer (no internet needed)

---

### üîó ACCESSING FROM ANOTHER COMPUTER

**Same network:**
1. Find IP: Run `ipconfig` (Windows) or `ifconfig` (Mac/Linux)
2. Look for: 192.168.x.x or 10.0.x.x
3. Other computer: Go to `http://YOUR_IP:8501`
   - Example: `http://192.168.1.100:8501`
